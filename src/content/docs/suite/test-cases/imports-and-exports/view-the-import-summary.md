You can use the Import Summary feature to view the status and details of your import and verify that the process completed as expected. This article explains how to view the import summary. 

---

> <p id="prerequisites">Prerequisites</p>
> 
> - Ensure you have imported artefacts from the current project to the destination project.

---

## **Steps to View the Import Summary**

1. From the left navigation bar, go to **Settings > Imports**.
   ![Imports](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_import_summary_1.png)

2. In the **Imports** page, click the status of the import you want to view. 
   ![Status](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_import_summary_2.png)

3. In the **Import Summary** dialog, verify the details, and click  **OK, Got it**. 
   ![Import Summary](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_import_summary_3.png)
