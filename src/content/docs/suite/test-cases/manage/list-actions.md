You can efficiently manage and organise your test cases in Testsigma through the Test Case List Actions. The Test Case List Actions allows you to easily search, sort, filter, and save your test cases.

---

## **Search Test Cases**

From the left navigation bar, go to **Create Tests > Test Cases** and click the list view icon. 
![list view](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_1.png)

Click inside the **Search** bar in the top right corner of the **Test Case List** page and type the name of the Test Case you want to find. As you type, Testsigma will show you matching test cases. View the test case you want by clicking on it in the search results 
![Search-Test-Cases](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_2.png)

---

## **Sort Test Cases**

Click the **Sort by** option in the menu bar on the **Test Cases List** page and arrange your test cases based on criteria such as **Title**, **Created Date**, or **Updated Date**. Sorting will help you prioritise and manage your test cases easily. 
![Sort-Test-Cases](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_3.png)

---

## **Filter Test Cases**

1. Click on **Show Filters** in the menu bar on the **Test Cases List** page.
![Show Filters](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_4.png)

2. Select **Add Filter** from the drop-down list. Choose from the following filtering options: **Status**, **Priority**, **Last run result**, **Added to Test Suite**, **Assignee**, **Created By**, **Created Date**, **Updated Date**, **Reviewer**, **Test Case Type**, **Labels**, **Requirement** and **Requirement Type**.
![Filter Test Cases](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_5.png)

You can filter Test Cases using user-defined custom and default fields. You can modify the filtering options by selecting **Add Filter** from the drop-down list and using multiple filters to obtain granular results.

---

## **Save Test Case Filter**

1. Click **Show Filters** in the menu bar on the **Test Cases List** page.
2. Click **Add Filter** to open the drop-down list and select the desired filters.
3. Click **Save Filter** will make the drop-down list appear for saving the current filter, and select **As New**.
![As new](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_6.png)

4. In the **Save filter config as** dialog, enter a name, select the **Mark as Public** checkbox if needed, and click **Save**. 
![Save Filter](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_7.png)

5. Click **Reset** to discard the current filter.
![Reset](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_8.png)

:::note[NOTE]
- Expand **Saved Filters** in the menu bar on the **Test Cases List** page to open the drop-down list. In **Custom Filters**, locate the saved filter, select the filter and mouseover to the filter, and click the **Edit** button next to the filter name to edit it. To delete the saved filter, click the **Delete** button. ![Save Test Case Filter](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_9.png)
:::

---

## **View Saved Test Case Filters**

1. Click on **Saved Filters** in the menu bar to open the drop-down list on the **Test Cases List** page.

2. Find the saved filter within **Custom** or **Predefined Filters** and choose it to view the filtered version. 
![View Saved Test Case Filter](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_10.png)

---

## **Update Saved Test Case Filter**

1. In **Test Case List** Page, expand **Saved Filters** and click on a filter.
![Saved Filters](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_11.png)

2. Click **Show Filters** to view all the existing filters and add the required filters. 
![Show filters](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_12.png)

3. Once you have added the filters, expand **Save Filter** and click **Replace Existing**. 
![Replace](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_13.png)

4. In the **Update filter config** dialog, edit the name, and click **Update**. 
![Update](https://s3.amazonaws.com/website-static-docs.testsigma.com/new_images/projects/Updated_Doc_Images/update_list_actions_14.png)

:::note[NOTE]
If you wish to discard the current filter, click **All** in the menu bar on the Test Cases List page.
:::
