## New features

**Agentic AI Home** — A unified dashboard for Gen AI and autonomous test automation flows.

**QTest Integration** — Flexible, customizable integration for test management and result syncing.

**Cross-Project Import** — Move test cases and assets between projects of the same application type.

**Export JSON** — Export all project artifacts in a single JSON for audit, compliance, or migration.

**SSO Login** — SSO authentication for secure access.

## Improvements

**Recorder Experience** — Onboarding infographics, scroll support, and a refactored Web Recorder for performance and stability.

**Step-Level Accessibility** — In-house accessibility checks at the step level for improved accuracy.

**Visual Testing Upgrades** — Thresholds, region selection, and AI-powered root-cause analysis to reduce false negatives.

**Immutable Audit Logs** — Exportable, searchable audit logs for compliance and traceability.

**Salesforce Flow Generation** — Automated test case generation for Salesforce flows and workflows.
