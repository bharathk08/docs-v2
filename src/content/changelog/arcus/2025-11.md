## New features

**Testsigma ↔ Test Management integration** — Link manual test cases to automated test cases in Testsigma, automate Test Management cases with Testsigma Agents, connect test plans, trigger automated executions from Test Management, and sync results back automatically — a unified flow from manual design to automation, execution, and reporting.

## Improvements

**Custom field filters on the Test Cases page** — Filter test cases by any custom field.

**Custom field filters in dynamic runs** — Combine custom fields with existing filters when creating dynamic runs to target regression, smoke, or release cycles precisely.

**Sorting on Test Cases and Test Runs** — Sort by title, order, or priority on both the Test Case and Test Run pages.
