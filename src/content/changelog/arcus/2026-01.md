## New features

**Read-only user role** — A view-only role across Test Management. Read-only users can see everything but cannot create, edit, delete, or execute any test assets.

**Confluence integration (PRD → test cases)** — Connect Confluence and convert PRDs and Confluence pages directly into test cases.

**Public shareable report links** — Share any report through a public link so stakeholders without a Test Management account can view it.

**Dataset selection for test cases & runs** — Parameterize test cases using datasets, and pick the dataset you need when creating a test case or executing a run.

**Requirement traceability report** — Generate traceability reports across one or more requirements with a custom report name, including charts and a full traceability matrix.

## Improvements

**Run history in test-case details** — Test cases now show their run history right on the details page, making it easier to track execution results across releases.

**Multi-region data centers** — EU and India data centers are now available alongside the existing US region to help meet regional compliance and data-residency requirements.
