## New features

**Review Management at the project level** — A structured review workflow for test cases: QA users submit cases for review, and reviewers approve or request rework with comments.

**Notifications & collaboration integrations** — Configurable notifications for key test events, delivered to Slack and Microsoft Teams for assignments, status changes, review updates, and more.

**Comments on test cases** — Inline commenting on test cases for discussion, feedback, and clarifications, including @-mentions for teammates.

**Dark theme** — Switch Test Management to a dark theme from settings.

## Improvements

**Dashboard enhancements** — The dashboard now surfaces automation coverage, test-case type distribution, and review-related actions for better visibility into test quality.

**Jira two-way sync improvements** — Backlog items are fetched automatically into sprints, test runs are paginated on Jira, and the two-way sync UI has been refreshed.

**Bulk edit for custom fields** — Update custom fields across many test cases at once.

**Preconditions in step templates** — Add preconditions directly within step templates in your test cases.

**Attachments in the Test Run API** — Upload logs, screenshots, and other evidence through the test-run result APIs.
