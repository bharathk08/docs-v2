## New features

**AI-based CSV field mapping** — Imports auto-map your CSV headers to Test Management fields by analyzing headers and data samples. Review the suggestions, override anything manually, and import in one click.

## Improvements

**Detailed import reports** — Every import now returns a full breakdown: total test cases processed versus successful, failed, and skipped, with record-level failure reasons and clear skip logic to help you clean data faster.
