## New features

**Copilot: Run from Any Step** — Start execution from any step, with no need to re-run the full setup prefix when debugging a late-failing step in a long test.

**Copilot: Pause & Resume** — Pause a live run instantly without pre-set breakpoints; the button dynamically switches between Resume and Pause.

**Edge & IE Mode Support** — Copilot now runs inside Microsoft Edge and Edge IE Mode, so enterprise teams with Edge-restricted apps no longer need workarounds.

**Atto: Conversational Task Control** — Create, edit, and remove test tasks via natural language prompts before and after browser sessions.

**Atto: Confluence Integration** — Atto connects to your Confluence workspace to pull documentation and requirements as context for test generation.

**Auto-Heal: Shadow DOM & iFrames** — Auto-Healing now detects and heals elements inside Shadow DOM and iFrame-embedded content.

**Mobile: Copilot & Atto Live Editor** — Full AI-assisted execution and Atto Live Editor extended to iOS and Android, with complete feature parity with the web editor.

**New Viewer Role** — Read-only Viewer role for stakeholders and auditors — full visibility without any edit access.

## Improvements

**Analyzer Agent in Live Editor** — Real-time failure insights directly in the editor; traces upstream steps and highlights the most probable root cause.

**Copy, Paste & Clone Steps** — Copy and clone individual or multiple steps across test cases; clone directly below or at any position.

**WCAG Accessibility Coverage +30%** — 30%+ of the latest WCAG success criteria now covered across Web, iOS, and Android.

**XRay: All Issue Types** — All XRay issue types (default and custom) now supported for AI-generated test context, not just "Test".

**TestRail: Multiple IDs per Test Case** — Link multiple TestRail IDs to a single Testsigma test case; many-to-one mapping was previously limited to one ID.

**Persistent Session History & Audit Log** — Append-only history log per Atto learning session tracks every task completion, pause, resume, and human response for full auditability.
