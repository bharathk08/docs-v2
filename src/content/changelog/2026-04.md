## New features

**Copilot: Unified Execution Platform** — Terminal is now Copilot: a single interface for execution, debugging, and agent interaction, with faster, more stable launches.

**WCAG 2.2 Accessibility Testing** — Full WCAG 2.2 testing with 30 criteria for Web and 18 each for Android and iOS.

**SOAP API Test Generation** — Atto can now generate SOAP API tests, with Excel (XLSX) support and Figma frame search.

**GitLab Integration** — Create bugs directly from failed tests in GitLab.

**Flutter Support** — Appium Flutter Integration Driver with UI element support for Flutter apps.

## Improvements

**Smarter Test Generation** — AI updates existing tests instead of duplicating them, with pre-save validation and real-time debugging of failures.

**Intelligent Auto-Heal** — Validates new locator changes against historical patterns before applying them.

**AES-256 Encryption** — Upgraded from DES, with API key expiration, SSL/TLS validation, and encrypted runtime variables.

**Live Editor & Recorder** — Redesigned Live Editor plus Shadow DOM and iFrame capture in the Recorder.

**Integration Enhancements** — Better Jira user search, Azure DevOps project filtering, and Zephyr custom-field support.
